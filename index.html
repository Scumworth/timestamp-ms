FCC API Basejump: Timestamp Microservice

User Stories

  1. I can pass a string as a parameter and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not conain a date or Unix timestamp, it returns null for those properties.

Example Usage

  • https://vast-journey-21759.herokuapp.com/October%2031,%201999
  • https://timestamp-ms.herokuapp.com/941342400

Example Output

{"unix":941342400,"natural":"October 31, 1999"}
